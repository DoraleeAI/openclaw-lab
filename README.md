# OpenClaw Lab

Node.js automation agent — a playground for OpenClaw experiments.

## Requirements

- Node.js >= 20

## Quick Start

```bash
npm start      # run once
npm run dev    # run with --watch (reloads on file changes)
npm test       # run node:test suite
```

## Layout

```
src/
  index.js     # entrypoint
```
